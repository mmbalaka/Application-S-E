# Application S&E — Lumière du Soleil

Application de **Suivi & Évaluation (S&E)** : dashboard « Lumière du Soleil ».

## Contenu du dépôt

| Dossier | Description |
|---------|-------------|
| [`dashboard/`](dashboard/) | Le projet du dashboard (écrans `.dc.html`, design system `_ds/`, exports). |
| [`dashboard-handoff/`](dashboard-handoff/) | Version « handoff » / livraison, avec son `README.md` et le sous-dossier `project/`. |

## Écrans principaux

- Connexion et accueils
- Dashboard principal
- Liste des projets / Fiche projet
- Calendrier Gantt
- Indicateurs et cibles
- Formulaire de rapport

Les fichiers `*-print.*` sont les versions destinées à l'impression / export PDF.
